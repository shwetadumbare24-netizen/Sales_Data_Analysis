# Sales Data Analysis using Pandas

## Objective

The objective of this project is to analyze sales data using Python and Pandas. The project demonstrates data loading, cleaning, filtering, grouping, and generating meaningful insights from a CSV dataset.

## Technologies Used

* Python 3
* Pandas
* Matplotlib

## Dataset

The dataset contains product sales information with the following columns:

* Product
* Category
* Sales

## Features

* Load CSV data using Pandas
* Data cleaning and preprocessing
* Filter records based on sales values
* Group data by category
* Generate summary statistics
* Visualize sales data using a bar chart

## Project Workflow

1. Load the CSV dataset.
2. Check and clean missing values.
3. Filter products with sales greater than a specific value.
4. Group data by category.
5. Calculate total sales by category.
6. Generate insights from the dataset.
7. Create a graphical visualization.

## Sample Insights

* Highest Sales Category: Electronics
* Total Sales calculated from the dataset
* Category-wise sales comparison

## Conclusion

This project demonstrates the use of Pandas for data analysis and Matplotlib for visualization. It helps in understanding how data can be processed, analyzed, and transformed into meaningful business insights.
